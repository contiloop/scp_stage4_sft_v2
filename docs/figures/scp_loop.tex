\caption{\textbf{One subset of the SCP loop.}
Steps (1)--(3) are the \emph{probe \& generate phase}: the current base
model $M_t$ first generates Korean translations under high-temperature
sampling ($Q_1$, $T{=}1.1$). A \emph{temporary} collapse LoRA is then
SFT-trained on $(x,\,\texttt{mt\_q1})$ as pseudo-labels and applied to
regenerate deterministic outputs ($Q_2$).
Steps (4)--(8) are the \emph{score, correct \& update phase}: a
reference-free QE score $S_i = f(Q_1, Q_2)$ measures both baseline
difficulty and collapse sensitivity, and samples below threshold $\tau$ are
flagged as fragile; the collapse LoRA is unloaded so subsequent stages run
on a clean base; only the fragile samples are sent to the external LLM for
correction-style relabelling; the base model is then SFT-updated in place
on the corrected pairs $(x, y^{*})$, and the same $M_t$ slot becomes the
input for the next subset.}
