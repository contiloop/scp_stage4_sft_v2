\caption{\textbf{SCP Stage 4 SFT pipeline.}
\textbf{(A)} Monolingual English datasets are normalized with tokenizer-aware
length splitting and partitioned into train/eval/OOD artifacts.
\textbf{(B)} Each subset $i$ runs the SCP loop in a U-shape: along the top
lane the current base $M_t$ first generates high-temperature $Q_1$, then a
\emph{temporary} collapse LoRA is trained on $Q_1$ pseudo-labels and used
for greedy $Q_2$. Along the bottom lane the reference-free QE score
$S_i = f(Q_1, Q_2)$ selects fragile samples; the collapse LoRA is unloaded
so the external LLM correction and the in-place SFT update of $M_t$ run on a
clean base. The dashed arrow indicates that the updated $M_t$ feeds the next
subset.
\textbf{(C)} After each subset, the updated $M_t$ is evaluated on a held-out
OOD set with reference-based MetricX-24, BLEU, and chrF, logged on the
\texttt{subset\_idx} axis.}
\label{fig:scp-pipeline}
